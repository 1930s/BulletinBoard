# BulletinBoard

## Requirements

- iOS 9 and later
- Swift 4

## Installation

BulletinBoard is available via CocoaPods and Carthage.

## CocoaPods

To install BulletinBoard using [Carthage](https://github.com/Carthage/Carthage), add this line to your `Cartfile`:

~~~ruby
pod "BulletinBoard"
~~~

## Carthage

To install BulletinBoard using [Carthage](https://github.com/Carthage/Carthage), add this line to your `Cartfile`:

~~~
github "alexaubry/BulletinBoard"
~~~
