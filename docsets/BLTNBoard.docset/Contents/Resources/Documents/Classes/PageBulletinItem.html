PageBulletinItem

open class PageBulletinItem: BulletinItem

A standard bulletin item with a title and optional additional informations. It can display a large action button and a smaller button for alternative options.

You can override this class to customize button tap handling. Override the actionButtonTapped(sender:) and alternativeButtonTapped(sender:) methods to handle tap events. Make sure to call super in your implementations if you do.
